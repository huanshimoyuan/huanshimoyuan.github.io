CSS3选择器

结构性伪类选择器

  1. :root 根选择器

    意义：匹配标签所在文档的根元素，在HTML文档中，根元素始终是<html>

         例如：
     :root{background:orange;}
     等同于
     html{background:orange;}

  2. :not 否定选择器

    意义：选择除某个元素之外的所有元素


     例如:

    style设置:
div p:not([class="not"]){
    border:1px solid red;
}
html文件:
&lt;div&gt;
    &lt;p class="not"&gt;不选择&lt;/p&gt;
    &lt;p>选择我&lt;/p&gt;
    &lt;p>选择我&lt;/p&gt;
&lt;div&gt;


    选择除去具有class=”not”属性的所有p标签，设置1px的实线红色边框。

  3. :empty 空选择器

    意义：选择没有任何内容的元素，空格都不能有


     CSS代码:

    p{
    background:orange;
    height:30px;
}
p:empty{
    display:none;        
}

    html代码:

    &lt;p&gt;这里有一个段落&lt;p&gt;
&lt;p&gt; &lt;p&gt;
&lt;p&gt;&lt;p&gt;


    选中一个空标签，将其隐藏起来。

  4. :target 目标选择器

    意义：用来匹配文档中的url的某个标识符的目标元素（这个属性我也不知道该用在什么地方，此事例摘自：慕课网）

     CSS代码:
     .menuSection{
         display:none;
     }
     :target{ /*这里的:target就是指id="brand"的div对象*/
         display:block
     }
 html代码：
     <h2><a href="#brand">Brand</a></h2>
     <div class="menuSection" id="brand">
         content for Brand
     </div>

    事例说明：当点击Brand（a标签）时，会显示一个隐藏文本
