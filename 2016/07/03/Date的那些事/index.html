Date的那些事

Date介绍

Date()语法可以帮助我们获取一些时间或者设置时间

下面介绍一部分语法，用来获取日期、时间

1) 获取当前时间

    var time = new Date();
    console.log(time);

定义变量time来存放获取到的现在的时间，这里获取的时间包括年、月、日、小时、分钟、秒和星期几

在Date后面的括号中写入日期，则表示设置时间，括号为空表示获取当前的时间

2) 获取年份 getFullYear()

    var year = new Date();
    var timeY = year.getFullYear();        //获取year的年份

如果在Date中设置时间，则获取到的是设置的年份，不设置，则表示获取当前时间的年份

3) 获取月份 getMonth()

    var year = new Date();
    var timeM = year.getMonth();

有人试着打印获取的时间会发现，月份会比现实中的月份小1，那是因为，这里的月份取值范围是0~11，0表示一月，1表示2月，以此类推

4) 获取月份中的日期(几号) getDate()

    var year = new Date();
    var timeD = year.getDate();

这里是获取当前的日期，即今天是几号，返回值就是几

5) 获取小时(24小时制) getHours()

    var year = new Date();
    var timeH = year.getHours();

返回值为当前时间的小时

6) 获取分钟 getMinutes()

    var year = new Date();
    var timeMi = year.getMinutes();

返回值为当前时间的分钟

7) 获取秒

    var year = new Date();
    var timeS = year.getSeconds();

返回值为当前时间的秒数

8) 获取毫秒 getMilliseconds()

    var year = new Date();
    var timeS = year.getMilliseconds();

返回值为当前时间的毫秒数

9) 获取时间戳 getTime()

    var year = new Date().getTime();

这里表示：当前时间距1970年1月1日的0时0分0秒有多少毫秒

我们一起来做一个倒计时，里约奥运越来越近了，我们来实现一下

    
    
        

            
距离里约奥运会还有:

            

                天小时分钟 秒

下面是我的代码:

    CSS样式:
    <style>
            *{
                margin:0;
                padding:0;
            }
            .wrap{
                width:600px;
                height:160px;
                margin:50px auto;
                background:dodgerblue;
                padding:15px;
                position: relative;
            }
            .wrap .text{
                position: relative;
                left:0;
                top: 0;
                font-size:20px;
                color:#FFFFFF;
            }
            .wrap .text span{
                font-size:30px;
            }
            .times {
                position: absolute;
                left:50%;
                top:50%;
                font-size:20px;
                color:orangered;
                transform: translate(-50%,-50%);
            }
            .times span:nth-child(2n-1){
                color:#fff;
                font-size:35px;
            }
        </style>

        HTML代码:
        <div class="wrap">
            <h1 class="text">距离<span>里约奥运会</span>还有:</h1>
            <p class="times">
                <span class="days"></span>
                <span>天</span>
                <span class="hours"></span>
                <span>小时</span>
                <span class="minutes"></span>
                <span>分钟 </span>
                <span class="seconds"></span>
                <span>秒</span>
            </p>
        </div>

        JS代码:
        <script>
        var wrap = document.querySelector(".wrap");
        var days = document.querySelector(".wrap .times .days");
        var hours = document.querySelector(".wrap .times .hours");
        var minutes = document.querySelector(".wrap .times .minutes");
        var seconds = document.querySelector(".wrap .times .seconds");

        times();
        function times(){
            setInterval(function(){
                var timeN = new Date().getTime();
                var timeTar = new Date(2016,7,6,7,0,0).getTime();
                var secs = parseInt((timeTar-timeN)/1000);    //将毫秒转换为秒
                var miao = secs%60;
                var fen = (secs-miao)/60%60;
                var shi = (secs-miao-fen*60)/3600%24;
                var tian = (secs-miao-fen*60-shi*3600)/86400;
                days.innerHTML = tian;
                hours.innerHTML = shi;
                minutes.innerHTML = fen;
                seconds.innerHTML = miao;
            },1000)
        }

    </script>
