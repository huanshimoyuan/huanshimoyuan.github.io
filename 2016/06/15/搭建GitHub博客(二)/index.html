搭建GitHub博客（二）

上回，已经安装了nodeJS,接下来开始安装hexo（注意输入语句中的空格，不要落下，接下来不再提示）

– 安装hexo

  1. 在终端中输入sudo npm install hexo -g回车

  2. 输入完成会提示输入密码，输入时，密码不会以字符的形式显示，可直接输入，回车，如果密码没有错误，需要等待一段时间，终端会自动下载并安装

  3. 验证hexo是否安装成功，终端输入hexo回车

– 本地创建博客文件目录

  • 在自己喜欢的位置新建文件夹（可通过鼠标右键或者终端进行操作），命名为blog（文件名可以随便设置，建议使用英文，以字母开头）

  • 打开终端，定位到新建文件夹的位置，输入hexo init进行文件夹的初始化,完成后，文件夹中会出现下面的文件，其中public文件夹没有，没有关系，最后完成博客发布，会出现。

  • 启动hexo服务，进行验证。终端输入hexo server回车,会出现一个本地连接，可以在浏览器中打开，看到的就是一个本地博客，按下ctrl+c退出hexo server。

– 建立GitHub账号仓库

  • 登陆账号，如果首页如下图，请点击Star a project

    如果不是上面的页面，请在页面中找到下图所示内容，点击New repository

    跳转到如下页面，按照图中步骤操作

  • 接下来就是比较关键的获取ssh-key，(定位在blog目录下)在终端中输入ssh-keygen连续回车三次，中间会有提示输入密码，不用输入，直接回车就可以。如下图所示：

  • 输入vim ~/.ssh/id_rsa.pub回车，按下图操作，复制完成直接输入:q退出

  • 打开之前申请GitHub账号的页面，选择SSH and GPG keys选项，点击New SSH key,粘贴ssh-key在文本框中，保存

  • 接下来需要进行GitHub与本地机器的验证，会用到git工具，安装Xcode软件的，不需要另外安装(这里将不再演示安装过程)

敬请期待!
