windows系统搭建GitHub博客

所需工具：

  1. github账号（去官网注册，并创建仓库,过程可参考之前的博客）
  2. git（可以去百度搜索下载）
  3. nodejs（官网下载即可）

注：下载软件的版本需和系统保持一致，不同软件版本的安装与使用可能会出现差异，请尽量与本教程保持一致，安装过程中，没有特殊说明的，提示输入密码，请输入管理员密码 也会有对话框弹出：输入用户名和密码，这里输入的时github账号的用户名和密码（下面不做说明，主要是忘记截图了）

本教程部分资料来自: 教程

安装Git

  1. 打开安装包，点击next

  2. 这里选择对应的windows选项，如下图：

  3. 等待安装完成

安装nodejs

安装过程没什么需要修改的，路径可根据自己的喜好选择，一直下一步就可以了

验证软件正确安装

同时按下Win键和R键（或者点击“开始菜单”->点击“运行”）,打开运行窗口,输入“cmd”
然后输入以下命令，结果与下图相同则说明安装正确，若不正确可以卸载软件重新安装
git –version
node -v
npm -v

注：命令输入完成之后需要按下回车，下面不再描述，输入完成后，自己按回车

安装hexo

注：我自己在安装过程中,有部分内容和图片不符,不过没有关系,输入命令行的时候,不要出现错误,应该都可以安装成功

  • 在电脑上新建一个blog文件夹,该文件夹用于存放你的博客文件,打开文件夹（既：进入文件夹）,然后右键单击选择“Git Bash”

  • 大家估计都有被“墙”的经历，安装hexo为了避免出现类似情况，使用淘宝NPM镜像,输入命令等待安装完成:npm install -g cnpm –registry=https://registry.npm.taobao.org保留代码中的空格

  • 使用npm安装hexo,输入命令:cnpm install -g hexo-cli等待hexo安装完成

  • 安装过程出现WARN不用理会，完成之后继续输入:cnpm install hexo –save
    安装完成后，输入:hexo -v
    会显示版本信息

注：代码的输入都是在blog文件夹下操作的(包括上面的操作)

  • 初始化博客文件夹
    输入:hexo init

等待初始化完成，完成后你会发现在文件夹中会生成一些文件

  • 安装生成器
    输入:cnpm install

  • 输入:hexo s -g
    以后要在本地运行博客只需要输入该命令即可

  • 打开浏览器，输入localhost:4000,就可以在本地看到你的个人博客了

  • 停止运行，按住Ctrl+c即可停止运行hexo

博客相关配置

  1. 配置信息
    打开blog文件夹，使用文本编辑器打开-config.yml文件，进行修改

  2. 配置ssh-key

    作用：使本地的git项目与远程的github建立联系

    a. 生成新的ssh-key
    git工具依旧在blog目录下打开，输入:ssh-keygen -t rsa -C “邮箱地址”
    系统提示（如下）：

    `

    ssh-keygen -t rsa -C “邮件地址@youremail.com”

    Generating public/private rsa key pair.
    Enter file in which to save the key (/Users/your_user_directory/.ssh/id_rsa):直接回车
    Enter passphrase (empty for no passphrase):再次回车
    Enter same passphrase again:再回车

    `

    注意：输入密码的时候没有*字样的，你直接输入就可以了
    最后看到这样的界面，就成功设置ssh key了：

    b. 添加SSH Key到GitHub

    在本机设置SSH Key之后，需要添加到GitHub上，以完成SSH链接的设置。

    * 1. 输入:vim ~/.ssh/id_rsa.pub
     弹出一串key，复制下来，然后直接输入":q"退出（""不需要输入）
     (或者：打开本地C:\Documents and Settings\Administrator.ssh\id_rsa.pub文件。此文件里面内容为刚才生成人密钥。如果看不到这个文件，你需要设置显示隐藏文件。准确的复制这个文件的内容，才能保证设置的成功。)

* 2. 登陆github系统。点击右上角的 Account Settings—->SSH Public keys —-> add another public keys

* 3. 把你本地生成的密钥复制到里面（key文本框中）， 点击 add key 就ok了
     注：此步骤有不清楚的地方请参考我的文章--搭建GitHub博客

    c. 验证git身份信息

    输入:git config --global user.name "你的用户名"
输入:git config --global user.email "你的邮箱"
这两步输入完成后不会有任何提示

    d. 安装hexo git插件

    输入:cnpm install hexo-deployer-git --save
等待安装完成

接下来就可以写博客了

写博客

  1. 在git工具中把目录切换到blog/source/_posts

    输入:hexo new 文章名称(注：图中的””可以不输入,直接输入文件名称即可，也不需要输入文件后缀)

    用编辑器打开新建的 文章名称.md文件，编写完成后保存

    然后把目录定位到blog文件夹下，输入:hexo server

    会显示一个本地的网址，可以用浏览器打开，看一下博客的显示状况，没有问题就可以发布到github上了，退出hexo server按下Ctrl+c

    输入:hexo g

    会自动编译网页，等待完成

    输入:hexo d

    等待本地网页的上传.

上传完成后，你就可以使用自己的域名浏览自己的博客了。既:用户名.github.io
