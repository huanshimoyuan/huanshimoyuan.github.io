lesscss简单语法

变量

定义变量：定义一个名为color的变量，值为red

格式：@color:red;

使用方法：

html代码:
<div class="setcolor"></div>

less代码:
@color:red;        <!--定义一个变量color-->
.setcolor{
    color:@color;    <!--调用变量-->
}

编译后的css代码:
.setcolor{
    color:red;
}

混合

混合可以将一个定义好的class A引入到另一个class B中，实现class B继承class A中的所有属性。 

html代码:
<div class="transf"></div>

less代码:
    <!--定义一个类(class)        .transf(定义函数,给定初始值)-->
    .tranf(@translateX:0px;@translateY:0px;@translateZ:0px;@rotate:30deg){
    -webkit-transform:translate3d(@translateX,@translateY,@translateZ) rotate(@rotate);
    <!--这行代码其实就是transform的简写方式,只是把数值换成了变量-->
    }

    .transf{
        width:100px;
        height:100px;
        background:blue;
        transition:all 3s ease 0s;
    }
    .transf:hover{
        .tranf(100px;-200px;0;360deg);        <!-- 调用类-->
    }

编译后的css代码:
    .transf{
        width:100px;
        height:100px;
        background:blue;
        transition:all 3s ease 0s;
    }
    .transf:hover{
        -webkit-transform:translate3d(100px,-200px,0) rotate(360deg);
    }

嵌套

类似C语言的程序嵌套，标签选择器也是可以进行嵌套的，最大的好处就是减少代码（目前我还没有感受到，初学阶段，代码都不会太长）

less代码:
.box{
    h1{
        color:gray;
    }
    div{
        font-size:20px;
        color:blue;
        p{
            font-size:16px;
            font-weight:bold;
        }
    }
}

编译后的css代码:
.box h1{
    color:gray;
}
.box div{
    font-size:20px;
    color:blue;
}
.box div p{
    font-size:16px;
    font-weight:bold;
}
