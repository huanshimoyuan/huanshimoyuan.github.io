JS基础

数据类型

数据类型的分类

数据类型分类:number(数字类型)、string(字符串类型)、boolean(布尔类型)、array(数组类型)、object(对象类型)、undefined(未定义类型)、null(空类型)

    e.g:
        var a=100;                        //number类型
        var b="200";    or    b='200';    //string类型
        var c=true;                        //boolean类型
        var d=[1,2,3];                    //数组类型
        var e;                            //undefined类型(无初始值)
        var f=null;                        //空类型(注：不等于0)

数据类型的转换

数据类型的转换可以分为两种：显式类型转换和隐式类型转换

  1. 显式类型转换

    字符串转换为数值类型(string->number)

    JS提供了两个转换函数parseInt和parseFloat.第一个是将字符串类型转换为整数，第二个是将字符串类型转换为浮点数(即小数).只有对字符串类型使用有效.对其他类型使用返回的都是NaN(Not a Number).

    现在来剖析一下转换类型的过程：

    1.) parseInt()函数首先查看位置下标为0处的字符，判断它是否是个有效数字，如果不是，该函数返回结果NaN，停止继续判断；但如果该字符是有效数字，则按照上述过程继续判断下一位字符，直到发现非有效数字的字符停止，然后将非有效字符之前的字符串转换为数字，下面来看几个栗子:

         parseInt("one123");        ->    返回值:NaN;
     parseInt("123one");        ->    返回值:123;
     parseInt("123.123");   ->    返回值:123; 注:对于整数来说，小数点是无效字符
     parseInt("0xA");        ->    返回值:10;  注:这里是个16进制的数值

    2.) parseFloat()函数的处理过程和parseInt类似，不过，第一个小数点是有效字符，如果有第二个小数点，则被视为无效，判断停止，这个小数点之前的字符串被转换为数字。(注:使用这个函数的字符串必须是十进制的，不能使用八进制或者十六进制形式)

         parseFloat("22.3.5");    ->    返回值:22.3;
     parseFloat("0234.25");    ->    返回值:234.25;  注:八进制形式前面的0会被忽略
     parseFloat("0x25.5");    ->    返回值:0;

    3.) 使用强制类型转换

    使用强制类型转换可以访问特定的值，即使它是另外一种类型

    Number(需要转换的值); ->将给定的值转换为数字，可以是整数或者浮点数

    强制类型转换

    有三种强制类型转换，上面一不小心暴露了一种，请看下方:

         格式:
     Number(value);        ->转换为数值
     Boolean(value);        ->转换为布尔值
     String(value);        ->转换为字符串
     e.g:
         Boolean("");                ->false
         Boolean("hello world");        ->true
         Boolean(1);                    ->true
         Boolean(0);                    ->false
         Boolean(null);                ->false
         var a;
         Boolean(a);                    ->false    注:定义的a变量是undefined类型
     e.g:
         Number(false);                ->0
         Number(true);                ->1
         Number(undefined);            ->NaN
         Number(null);                ->0
         Number("12.12");            ->12.12
         Number("5.5.5");            ->NaN
         Number("a52");                ->NaN
         Number(100);                ->100
         Number(59.9);                ->59.9
     e.g:
         String(100);                ->"100"
         String(null);                ->"null"
         String(undefined);            ->"undefined"
         String(false);                ->"false"
         String(12.12);                ->"12.12"
         String(12.12.12);            ->报错

  2. 隐式类型转换

         e.g:
         var ch="12.12";        //定义一个变量，存储一个字符串
         var a=ch-0;            //定义一个变量，存储运算结果
         console.log(a);        //查看运算结果a为12.12
