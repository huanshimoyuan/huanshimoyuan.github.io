Tab切换

Tab切换效果的实现

首先，我来做个案例，先看一下什么是Tab切换

    

    

        

        
A

        
B

        
C

试着点击一下上面的按钮，看看有什么变化。

这个效果在很多网站上都有，使用的频率相当的高，首先我们需要来确定一下实现这个效果的步骤：

  1. 页面布局,元素的样式设置(CSS)
  2. 定义变量，用来查找元素
  3. 给元素添加点击事件，以便触发元素的变化
  4. 元素变化所需的方法

步骤已经确定，那我们就来按照上面的顺序编写代码:

First:

布局，随便你们，高兴就好，都是最基础的知识，下面就不作出解释了，注：代码一定不要只看不敲,文档的格式自己注意一些，不要排列错误，这里的顺序不等于文档顺序

我们现在需要三个按钮和三个div元素，按钮可以使用input来实现，代码如下:


    <div class="box">
        <input type="button" value="按钮A" class="btn">
        <input type="button" value="按钮B" class="btn">
        <input type="button" value="按钮C" class="btn">
        <div class="let color">A</div>
        <div class="let color1">B</div>
        <div class="let color2">C</div>
    </div>

接下来，要进行CSS的书写:

    <style type="text/css">
        .btn{
            background-color:gray;
            padding:3px 5px;
            margin:0 20px;
        }
        .let{
            width:100px;
            height:100px;
            background-color:green;
            text-align:center;
            line-height:100px;
            font-size:50px;
            color:#000;
            position:absolute;
            left:80px;
            top:30px;
        }
        .color{
            background-color:green;
            display:block;
        }
        .color1{
            background-color:pink;
            display:none;
        }
        .color2{
            background-color:blue;
            display:none;
        }
    </style>

Second:

定义变量，用来查找元素，这里使用的选择器功能很强大，以后会经常用到，多加练习吧，骚年！

这里为什么会需要变量呢？我们每次点击的按钮是不确定的，也是会变化的，使用变量存储他们的名称可方便我们获取元素

    <script type="text/javas">
        var btns=document.getElementsByClassName("btn");
        var lets=document.getElementsByClassName("let");

        btns[0].style.backgroundColor="gold";
        for(var i=0;i<btns.length;i++){
            btns[i].sub=i;
            btns[i].onclick=function(){
                for(var j=0;j<btns.length;j++){
                    btns[j].style.backgroundColor="";
                    lets[j].style.display="none";
                }
                this.style.backgroundColor="gold";
                lets[this.sub].style.display="block";
            }
        }
    </script>

这里需要进行一些说明:

定义了变量btns，用来获取三个按钮元素，定义变量lets来获取每个div元素；

然后给第一个按钮设置了一个背景颜色，这里需要注意，你可以尝试一下，将这个属性设置在CSS中，看看区别(此处有大坑，尝试一些才能更好的理解接下来的代码，不要偷懒哦)，设置在CSS中的属性，在script中无法获取，所以，我们需要将这些设置放在script中。

我们要做到点击按钮时，页面发生变化，就需要给按钮添加点击事件，我们可以使用for循环给每个按钮添加onclick事件，控制for循环的开始与结束就很关键了，btns可以理解为一个数组，用来存放一系列的元素名称，第一个元素下标为0，第二个元素下标为1，一次类推，定义一个变量i，初始化为0，和我们元素的下标开始位置相对应，我们获取变量btns的长度，用来进行判断，每循环一次，i自增1，直到不满足判断条件，跳出for循环。

大家应该有注意到btns[i].sub=i;，这句代码写在for循环中，sub是自己定义的一个属性，当执行for循环时，为每个元素添加sub属性，并赋值为i，即：i=0时,btns[0]=0;i=1时,btns[1]=1;i=2时,btns=2;注意这里的赋值，我给他们赋的值都是等于他们的下标，接下来我们要获取元素的下标，就可以通过获得自定义属性的值，来得到下标(还有，这里的自定义属性我也是在script中设置的，为什么没有写在CSS中呢，还记得上面提到的:在script中，无法获取CSS中的属性，更不要说，获取属性的值了)

必须要说的一点，在页面打开的时候，外层的for循环就已经执行完毕了，这里i的值始终为2，外层for循环的作用就是给每个btns元素添加点击事件

btns[i].onclick=function(){}大括号中的内容都是需要通过点击按钮来触发的，页面打开时，是不会执行的。

大括号中的for循环是用来清除样式的，每次循环的时候先清除所有元素的这些样式(主要是清除出去某个元素之外的元素属性不容易做，那我们就清除所有，然后再添加新样式)，用this获取点击的元素，并设置新的样式。

this的作用就是，我们点击某个元素，就会自动获取该元素，this.sub是用来获取点击的元素的sub属性值，这里等于元素下标值

这样，我们就实现了Tab切换效果，多加练习！
